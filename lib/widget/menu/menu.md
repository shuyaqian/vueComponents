
Menu组件
===
属性说明如下:

### Menu Attributes
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- |
| tabList | 菜单列表 | Array | — |  |
| hasBorder | 是否有左右边框 | boolean | — | true |
| hasFt | 是否有右侧按钮 | boolean | — | false |
| iconSrc | 图标路径 | String | - | - |
| selectedColor | 选中颜色 | string | - | - |
|  |  |  |  |  |
